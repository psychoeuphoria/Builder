批量处理照片以节省时间

标题 Batch Processing Photos to Save Time
讲师 Richard Harrington
时长 1时29分
发布日期 2021年5月12日

课程描述

图像通常需要相同的调整。例如，婚礼等活动是在同一天用相同的光线、相同的背景等拍摄的。如果你使用自动化批量处理照片，而不是煞费苦心地编辑每张照片，你可以节省大量时间并更快地获得相同的结果。本课程介绍了用于批量处理照片集的工作流程和工具。摄影师 Rich Harrington 首先解释了批处理可以做什么和不能做什么。然后，他介绍了四种不同的工具集来执行工作：内置于 macOS 和 Windows 中的本机应用程序；实用程序和基于 Web 的工具，例如 XnConvert、IffanView 和 QuickScale；专业级 Adob​​e 套件，包括 Photoshop 和 Lightroom；以及 Luminar AI 中提供的基于 AI 的功能。在此过程中，Rich 展示了如何创建以进一步扩展初始调整的好处。
