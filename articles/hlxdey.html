Android NDK

字符串

Java 字符串由 JNI 作为引用类型处理。引用类型不能直接用作原生 C 字符串。 由于 Java 字符串对象是不可变的， JNI 不提供任何修改 Java 字符串内容的函数。

新字符串

函数 NewString 用于创建 Unicode 字符串，NewStringUTF 用于创建 UTF-8 字符串。

jstring javaString;
javaString = (*env)->NewStringUTF(env, "Hello World!");

在内存溢出的情况下，返回 NULL。

将 Java 字符串转换为 C 字符串

函数 GetStringChars 可以将 Unicode 字符串和 GetStringUTFChars 可以将 UTF-8 字符串转换为 C 字符串。

const jbyte* str;
jboolean isCopy;
  
str = (*env)->GetStringUTFChars(env, javaString, &isCopy);
if (0 != str) {
    printf("Java string: %s", str);
  
    if (JNI_TRUE == isCopy) {
     printf("C string is a copy of the Java string.");
    } else {
     printf("C string points to actual string.");
    }
}

释放字符串

通过 GetStringChars 和 GetStringUTFChars 函数获取的 C 字符串需要在使用完后正确释放，否则会发生内存泄漏。


(*env)->ReleaseStringUTFChars(env, javaString, str);

数组

Java 数组由 JNI 作为引用类型处理。

新数组

New<Type>Array 函数可以从本机代码构造新的数组实例，<Type> 为 Int、Char、Boolean 等，例如 NewIntArray。

jintArray javaArray;
javaArray = (*env)->NewIntArray(env, 10);
if (0 != javaArray) {
    /* You can now use the array. */
}

与 NewString 函数相同，在内存溢出的情况下，New<Type>Array 函数将返回 NULL。

访问数组元素

// 获取 Java 数组部分的副本作为 C 数组
jint nativeArray[10];
(*env)->GetIntArrayRegion(env, javaArray, 0, 10, nativeArray);

// 提交从 C 数组到 Java 数组的更改
(*env)->SetIntArrayRegion(env, javaArray, 0, 10, nativeArray);

// 获取指向 Java 数组元素的直接指针
jint* nativeDirectArray;
jboolean isCopy;
  
nativeDirectArray = (*env)->GetIntArrayElements(env, javaArray, &isCopy);

// 释放指向 Java 数组元素的直接指针
(*env)->ReleaseIntArrayElements(env, javaArray, nativeDirectArray, 0);

// 基于给定 C 字节数组的新字节缓冲区
unsigned char* buffer = (unsigned char*) malloc(1024);
...
jobject directBuffer;
directBuffer = (*env)->NewDirectByteBuffer(env, buffer, 1024);

// 从 Java 字节缓冲区获取字节数组
unsigned char* buffer;
buffer = (unsigned char*) (*env)->GetDirectBufferAddress(env,
    directBuffer);

字段

获取字段 ID

public class JavaClass {
    /** Instance field */
    private String instanceField = "Instance Field";
  
    /** Static field */
    private static String staticField = "Static Field";
  
    ...
}

// 从对象引用中获取类
jclass clazz;
clazz = (*env)->GetObjectClass(env, instance);

// 获取实例字段的字段 ID
jfieldID instanceFieldId;
instanceFieldId = (*env)->GetFieldID(env, clazz,
    "instanceField", "Ljava/lang/String;");

// 获取静态字段的字段 ID
jfieldID staticFieldId;
staticFieldId = (*env)->GetStaticFieldID(env, clazz,
    "staticField", "Ljava/lang/String;");

// 获取实例字段
jstring instanceField;
instanceField = (*env)->GetObjectField(env, instance, instanceFieldId);

// 获取静态字段
jstring staticField;
staticField = (*env)->GetStaticObjectField(env, clazz, staticFieldId);
